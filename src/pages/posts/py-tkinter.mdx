## Basic setup

We are going to use Tkinter and also add TTkboostrap with provides some basic styling like it's counterpart. in the CSS world. This allows for the theme assignment. Not the aliasing so that ttkbootstrap overides ttk.

An `input frame` is specified covering the entire window. Then the `entry` widget and the `button` are placed inside the 'input frame'

```py
import tkinter as tk
from tkinter import ttk
import ttkbootstrap as ttk

# window
window = ttk.Window(themename = 'journal')
window.title('Demo')
window.geometry('300x150')
```

The function below is called when the button is pressed:

```py
def convert():
    mile_input = entry_int.get()
    km_output = mile_input * 1.61
    output_string.set(km_output)
```
The rest of the code is fairly self explanatory. Note
the built-in types `IntVar` and `StringVar` which are assgined using the `textvariable` keyword. 

```py
#title
title_label = ttk.Label(master = window, text = 'Miles per kilometer', font = 'Calibri 24 bold')
title_label.pack()

# input field
input_frame = ttk.Frame(master = window)
entry_int = tk.IntVar()
entry = ttk.Entry(master = input_frame, textvariable= entry_int)
button = ttk.Button(master = input_frame, text = 'Convert', command = convert)

entry.pack(side = 'left', padx = 10)
button.pack(side = 'left')
input_frame.pack(pady = 10)

# output
output_string = tk.StringVar()
output_label = ttk.Label(
    master = window,
    text= "Output",
    font = 'Calibri 24',
    textvariable= output_string)
output_label.pack(pady = 5)

# run
window.mainloop()
```

## Setup build for TS if needed later-But allow js for now

Modify the `compilerOptions` of the 'tsconfig.json' file. Simply make sure you have the `allowJs` option set to `true`


```json
{
  "files": [],
  "compilerOptions": {
    "allowJs": true
  },
}
```

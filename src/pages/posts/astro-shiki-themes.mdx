## Select the theme you want

First [select the theme](https://shiki.style/themes) you want or else find them in the [github repo](https://github.com/shikijs/textmate-grammars-themes/tree/main/packages/tm-themes)

Adapt the theme with the css as shown below:

```css
.astro-code {
  margin-top: .5em;
  border-radius: 7px;
  padding: .5em;
}
```

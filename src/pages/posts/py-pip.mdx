## Handling the requirements.txt file

To make a requirements.txt that shows all the Python packages required for your projected use:

```py
pip freeze > requirements.txt
```

Then if you need to install from a requirements.txt file from another repository you have forked then use the following:

```py
pip install -r requirements.txt
```

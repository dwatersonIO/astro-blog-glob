## Setup the plots

Notice how the .plot method has attributes for color, width and linestyle. If you selected a theme above then best to leave these black and let the theme decide on the colors and styles.

```python
plt.plot(ages_x, py_dev_y, color='#5a7d9a', linewidth="3", label='Python devs')
plt.plot(ages_x, js_dev_y, color='#adad3b', linewidth="3", label='Javascript')
plt.plot(ages_x, dev_y, color='b', linestyle='--',label='All devs')

plt.title('David test plot')

plt.legend()

plt.xlabel('ages')
plt.ylabel('salary')

plt.grid(True)

plt.tight_layout() # adjusts the padding for smaller displays

plt.show()
```

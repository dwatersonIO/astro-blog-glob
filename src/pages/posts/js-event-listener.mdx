Imagine the div below will be populated with a number of li elements, such as when adding items for a todo list app. If the outside container has a data
attribute like below it will be easy to query the contents.

```html
<div class="todo-list" data-list-display-container>
...
</div>
```
First we assign the div with that data attribute to a variable. Then the js code checks to see if the user clicked on li item.
If he did then certain actions are performed, in this case a variable (selectedListId) is assigned the value of the listID selected.

```js
const listsContainer = document.querySelector('[data-lists]')

listsContainer.addEventListener('click', e => {
    if (e.target.tagName.toLowerCase() === 'li') {
        selectedListId = e.target.dataset.listId
        saveAndRender()
    }
  })

```

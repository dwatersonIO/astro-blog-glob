## Create a virtual environment

```bash
python -m venv venv
```

## Activate virtual environment

*Windows*
```bash
venv\scripts\activate.bat
```   
*Linux*
```bash
source .venv/bin/activate
```
To deactivate a virtual environment one command used for all operating systems:
`deactivate`

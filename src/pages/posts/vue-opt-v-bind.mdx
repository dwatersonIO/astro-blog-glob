## Simple example using the v-bind

V-bind can be shorted to simply : as shown below.


```html
<body>        
    <div id="app">
        <p v-bind:class="paraClass">Paragraph style is red</p>
        <h3 :class="headingClass">Heading style is green</h3>
    </div>  

    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
    
    <script>
        const { createApp } = Vue
        const app = createApp ({
            data () {
                return {
                    paraClass: "red",
                    headingClass: "green"
                }
            },
        })
        app.mount('#app')
        </script>
    </body>
</html>

<style>
.red {
    color: red
}
.green {
    color: green
}
</style>

```
